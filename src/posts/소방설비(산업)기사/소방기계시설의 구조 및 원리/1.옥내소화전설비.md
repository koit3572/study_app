# 옥내소화전설비
---
## 개요
### 정의
- `옥내`에 `설치`하여 `화재발생` 시 `소화전함`까지 `배관`이 `연결`되어 있고 `소화전함 내`의 `방수구`로부터 `연결`된 `소방호스`의 `말단`에 `노즐`을 `연결`하여 `물`을 `방수하는 설비`
### 구성요소
1. `수원`
2. `가압송수장치`
3. `배관`
4. `송수구`
5. `방수구`
6. `옥내소화전함`
7. `제어반`
8. `전원`
9. `배선`
---
## 수원
### 수원의 양
|층수|수원의 양 공식|기호 및 단위|
|--|--|--|
|`29층 이하`|`Q = N * 2.6m³`|Q = `수원의 양[m³]` <br/> N = `가장 많은 층의 소화전 설치개수(최대 2개)`|
|`30층 이상 49층 이하`|`Q = N * 2.6m³`|Q = `수원의 양[m³]` <br/> N = `가장 많은 층의 소화전 설치개수(최대 2개)`|
|`50층 이상`|`Q = N * 2.6m³`|Q = `수원의 양[m³]` <br/> N = `가장 많은 층의 소화전 설치개수(최대 2개)`|
### 수원의 저장 방법
- `옥내소화전 설비`의 `수원`은 `산출`된 `유효 수량` 외에 추가로 `유효 수량의 1/3 이상`을 `옥상`에 저장
### 1/3을 옥상에 저장하지 아니하여도 되는 경우
1. `지하층만 있는 건축물`
2. `고가수조`, `가압수조`를 `가압송수장치`로 `설치`한 `옥내소화전설비`
3. `수원`이 `건축물`의 `최상층`에 `설치`된 `방수구`보다 `높은 위치에 설치`된 경우
4. `건축물`의 `높이`가 `지표면으로부터 10m이하`인 경우
5. `주펌프`와 `동등 이상`의 `성능`이 있는 `별도`의 `펌프`로서 `내연기관`의 `기동과 연동`하여 `작동`되거나 `비상전원을 연결`하여 `설치`한 경우
### 유효수량
---
- 펌프의 `유효수량`은 `소화펌프`와 `급수펌프`의 `후드밸브`의 `사이`를 의미한다.<br/>
  (이때 소화펌프의 후드밸브가 급수펌프의 후드밸브보다 `아래`에 있어야 한다.)
