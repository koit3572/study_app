# 소방안전관리자
---
## 기본 개념
1. `소방안전관리자`
    - 특정소방대상물의 각 `관리대상 부분`을 맡는 책임자
2. `총괄소방안전관리자`
    - 여러 부분 소방안전관리자(권원별 소방안전관리자)를 전체적으로 지휘·조정하는 사람
3. `권원`
    - 해당 `공간`을 `실제`로 `사용`하는 `운영주체`<br/>
      (소유자가1명이어도 임차인이 여러명이면 권원1개)
4. 하나의 특정소방대상물이란?
    - `하나의 건물`(`동`) 또는 `하나로 묶인 단지`
    - 건축물대장 `총괄표제부`가 있으면 `하나의 대상물`로 취급
    - `총괄표제부` 없이 각 동이 `표제부`만 있으면 각 동이 `별개`로 취급
---
## 총괄소방안전관리자가 필요한 조건
- 아래의 3개의 조건이 모두 충족해야 만족하는것으로 판단
1. `하나의 특정소방대상물`
2. `선임대상` 부분이 `2개 이상`
3. `권워(점유자)`이 `2명 이상`<br/>
  (`점유자`가 `1명`인경우 `무조건 제외`)
---
## 소방안전관리자 구조
1. `소방안전관리자 1명`
    - 거진 99%이상 이 구조에 해당한다
    - ex : `1명의 점유자` + `x개의 선임대상` or `x명의 점유자` + `1개의 선임대상`
      - 서류1(소방안전관리자 1명)
        - 관리권원분리 [   ]해당 [ √ ]해당없음
        - [   ]권원별소방안전관리자 선임 [ √ ] 소방안전관리자 1명 선임
        - 총괄소방안전관리자 [   ]해당 [ √ ]해당없음
2. `권원별소방안전관리자 x명` + `총괄소방안전관리자 1명`
    - `1명`이 `모든 역활`을 `겸직`할 수 있으며, `1명`이 하더라도 `역활수`만큼 `서류 제출` 필요
    - ex : `2명이상의 점유자` + `2개이상의 선임대상`
      - 서류1(권원별소방안전관리자)
        - 관리권원분리 [ √ ]해당 [   ]해당없음
        - [ √ ]권원별소방안전관리자 선임 [   ] 소방안전관리자 1명 선임
        - 총괄소방안전관리자 [   ]해당 [ √ ]해당없음
      - 서류2(권원별소방안전관리자)
        - 관리권원분리 [ √ ]해당 [   ]해당없음
        - [ √ ]권원별소방안전관리자 선임 [   ] 소방안전관리자 1명 선임
        - 총괄소방안전관리자 [   ]해당 [ √ ]해당없음
      - 서류3(총괄소방안전관리자)
        - 관리권원분리 [ √ ]해당 [   ]해당없음
        - [   ]권원별소방안전관리자 선임 [   ] 소방안전관리자 1명 선임
        - 총괄소방안전관리자 [ √ ]해당 [   ]해당없음

## 용도별 소방안전관리자 선임대상 기준
### 특정소방대상물 등급과 소방안전관리자 등급 관계
- `건물 등급` 산정 후 `소방안전관리자 등급`이 결정되는 구조
- 사실상 `건물 등급` 보다 `소방안전관리자 등급` `1단계` 높거나 `같`을 수 밖에 없는 구조
- 특정소방대상물 등급(건물 자체의 조건)
    1. `층수`
    2. `높이`
    3. `연면적`
    4. `위험물 저장량`
    5. `터널 길이`
- 소방안전관리자 등급(관리 난이도/설비수준)
    1. `설치된 소방시설 종류`
    2. `연면적`
    3. `가연성가스 저장량`
    4. `위험물 저장량`
    5. `화재안전기준에 따른 설비 의무`
### 실무상 따로 알아야 하는가?
- 건물 등급에 이미 `소방설비`, `위험도`, `면적`등이 `반영`되어 소방안전관리자 등급도 `자동`으로 따라오게 됨
- `제도 설계`상 `건물의 위험도`와 `사람자격능력`을 `구분`하기 위한것
